Zynova
Overview

Zynova is not just a traditional e-commerce platform. It is a smart marketplace that empowers sellers to create and innovate products quickly using Artificial Intelligence powered by the Google Gemini API. The platform also provides a smooth and modern shopping experience for users.

User Roles

Zynova supports two main types of users:

Shopper
Browse products across different categories
Search and filter items easily
Add products to the cart and place orders
Vendor
Manage a personal virtual store through a dashboard
Add, edit, and organize products
Track orders and monitor sales activity
AI Features

Zynova integrates Google Gemini to provide powerful AI-driven tools:

Product Generation

Sellers can enter a category (e.g., electronics, furniture), and the AI generates:

Product names
Descriptions
Suggested quantities
Creative product ideas
Image Generation

Using the Imagen model, the platform creates high-quality product images based on text prompts.

Description Enhancement

AI helps sellers improve product descriptions by turning simple keywords into engaging marketing content.

Technical Features
Vendor Dashboard: Full control over products, inventory, and order tracking
Shopping Cart System: Manage purchases, calculate totals, and handle multi-vendor orders
Subscription System: Products are visible only to vendors with an active subscription
Modern UI: Built with React and Tailwind CSS for a responsive and clean interface
Project Goal

The goal of Zynova is to simplify entry into e-commerce by removing the need for pre-made product content. With AI-powered product generation, sellers can launch stores faster and focus on creativity rather than content creation.
